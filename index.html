กรอกคะแนนนักเรียน

ระบบคิด: (คะแนนเก็บ ÷ เต็มเก็บ × 70) + (ปลายภาค ÷ เต็มปลายภาค × 30) = ร้อยละ → เกรด 0–4 • เลือก ร ได้เองกรณีงาน/คะแนนไม่ครบ • มส ระบบขึ้นให้เมื่อเวลาเรียนต่ำกว่าเกณฑ์ (ปลดได้ที่แท็บ ซ่อม มส)

🕒 เวลาเรียนรายคน

นับคาบจากตารางสอน × วันเปิดเรียน (วันที่มีเช็คชื่อหน้าเสาธง) • คาบที่ครูไม่ได้เช็ค ใช้ผลหน้าเสาธงแทน • รวมคาบซ่อมชดเชยแล้ว • ต่ำกว่า 80% = มส — เพิ่มคาบซ่อมจนถึงเกณฑ์เพื่อปลด มส

เลขที่ชื่อ-นามสกุล คาบทั้งหมดมาเรียน ซ่อมแล้วเวลาเรียน % สถานะ

📖 คู่มือการใช้งาน

ขั้นตอนการใช้งานโดยย่อ

  1. แอดมินตั้งค่า ข้อมูลสถานศึกษา (โลโก้ ชื่อโรงเรียน ปีการศึกษา 2569 ภาคเรียน ผอ./รองผอ. ลายเซ็น)
  2. นำเข้า รายชื่อนักเรียน ม.1/1–ม.3/2 (ดาวน์โหลดฟอร์มไปกรอกแล้วนำเข้า)
  3. นำเข้า ตารางสอน ของภาคเรียน → ระบบสร้างบัญชีครูและรายวิชาให้อัตโนมัติ
  4. ครูประจำวิชาตั้ง สัดส่วนคะแนน (เช่น 70:30) + หน่วยคะแนนเก็บ แล้วบันทึกคะแนน → ระบบตัดเกรด 0–4 / ร / มส ให้
  5. เช็คชื่อหน้าเสาธง/รายคาบ — ระบบนับเวลาเรียนรายวิชา ต่ำกว่า 80% ขึ้นธง มส (ซ่อมเวลาเรียนเพื่อปลดได้)
  6. สิ้นภาคเรียน: พิมพ์ใบรายงาน + ส่งออก Excel เข้า School MIS รายภาคเรียน

วิธี Deploy เป็น Web App (สำหรับครูที่นำโค้ดไปใช้ต่อ)

  1. สร้าง Google Sheet ใหม่ → Extensions → Apps Script → วาง Code.gs
  2. รันฟังก์ชัน INIT 1 ครั้งเพื่อให้สิทธิ์และสร้างชีตทั้งหมด
  3. Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone
  4. คัดลอก Web App URL มาวางในไฟล์ index.html ที่ตัวแปร window.WEB_APP_URL (บนสุดของไฟล์)
  5. อัปโหลด index.html ขึ้น GitHub Pages แล้วใช้งานผ่านลิงก์ของ GitHub Pages
