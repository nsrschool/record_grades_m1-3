กำลังเชื่อมต่อ...

💡 ข้อมูลถูกบันทึกบน Google Sheets ของคุณโดยอัตโนมัติ สามารถเข้าใช้งานได้จากทุกอุปกรณ์

อันดับ เลขที่ ชื่อ-นามสกุล หน่วยกิต GPA/GPAX รายงาน

📐 เกณฑ์การตัดสินผลการเรียน (8 ระดับ)

ตั้งแต่ %ถึง %เกรดความหมาย
เลขที่ เลขประจำตัว ชื่อ-นามสกุล ห้อง จัดการ

📋 ขั้นตอนการใช้งานโดยย่อ

  1. ไปที่ ⚙️ ตั้งค่าระบบ → ข้อมูลสถานศึกษา กรอกชื่อโรงเรียน/ผอ. อัปโหลดโลโก้และลายเซ็น แล้วบันทึก
  2. ไปที่ จัดการรายชื่อนักเรียน นำเข้ารายชื่อจากไฟล์ Excel หรือเพิ่มทีละคน
  3. ไปที่ 📚 รายวิชา เพิ่มรายวิชาแยกตามห้องและภาคเรียน (กรอกชั่วโมง ระบบคิดหน่วยกิตให้)
  4. ไปที่ ✍️ บันทึกผลการเรียน เลือกห้อง→ภาคเรียน→วิชา กรอก % ระบบแปลงเป็นเกรดอัตโนมัติ (เลือก ร/มส ได้)
  5. ไปที่ 🌟 ประเมิน บันทึกคุณลักษณะ อ่านคิดวิเคราะห์เขียน และกิจกรรมพัฒนาผู้เรียน
  6. ไปที่ 📊 รายงานสรุป ดู GPA/GPAX จัดอันดับ และพิมพ์สมุดรายงานประจำตัว

🚀 วิธี Deploy เป็น Web App

  1. สร้าง Google Sheet ใหม่ → เมนู ส่วนขยาย (Extensions) → Apps Script
  2. วางโค้ดใน Code.gs และสร้างไฟล์ HTML ชื่อ index แล้ววางโค้ด HTML
  3. กด Deploy → New deployment → เลือก Web app
  4. ตั้งค่า Execute as: Me และ Who has access: Anyone (หรือ Anyone with Google account)
  5. ครั้งแรกต้องกด อนุญาตสิทธิ์ (Authorize) Sheets + Drive
  6. คัดลอก Web App URL ที่ลงท้ายด้วย /exec ไปใช้งาน/แชร์

เมื่อแก้โค้ดภายหลัง: Deploy → Manage deployments → ✏️ Edit → New version → Deploy
